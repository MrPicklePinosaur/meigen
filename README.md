# 迷言語 / meigengo

</div>

## Running the parser

Currently input to the parser is taken from the command line:
```sh
cabal v2-run meigen -- <input>
```

## Why the name?

明言 (declaration), 名言 (famous quote), 迷言 (stupid saying), you can interpret it however you want.
